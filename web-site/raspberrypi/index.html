Raspberry Pi

Le Raspberry Pi est un nano-ordinateur monocarte à processeur ARM de la taille d'une carte de crédit

conçu par des professeurs du département informatique de l'université de Cambridge dans le cadre de la fondation Raspberry Pi.

 

Description

Les différents Modèles :

  • Modèle A
    1. Modèle 1A
    2. Modèle 1A+
    3. Modèle 3A+

  • Modèle B
    1. Modèle 1B
    2. Modèle 1B+
    3. Modèle 2B
    4. Modèle 3B
    5. Modèle 3B+
    6. Modèle 4B

  • Modèle Zéro
    1. Modèle Zéro W
    2. Modèle Zéro WH
